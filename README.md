## Troubleshooting

### GitHub Pages not deploying

1. Check Settings â†’ Pages â†’ Source is set to "main" branch
2. Wait 2-3 minutes for initial deployment
3. Check the Actions tab for build errors
4. Make sure `index.html` is in the root (not in a subfolder)

### Student benefits expired

1. Go to [GitHub Education](https://education.github.com/benefits)
2. Re-verify with your @ewu.edu email
3. Benefits usually renew within 24-48 hours

### Copilot not suggesting code

1. Make sure GitHub Student Pack is active
2. In VS Code, check that Copilot extension is installed and signed in
3. Try typing a comment describing what you want, then press Enter

### Figma education tier issues

1. Go to Figma â†’ Settings â†’ Account
2. Check if "Education" plan is active
3. If not, re-apply at [figma.com/education](https://figma.com/education)
